# Extracción, Limpieza y Análisis de Datos del Banco Mundial

## Descripción
Este repositorio contiene un cuaderno interactivo desarrollado para extraer, limpiar y analizar datos del Banco Mundial. El objetivo principal es demostrar el funcionamiento de un pipeline para obtener subconjuntos de datos a partir de los conjuntos disponibles en el catálogo del Banco Mundial.

## Indicadores Consultados
1. ODS #8 Desarrollo Económico Sostenible: [Sustainable Development Goals](https://datacatalog.worldbank.org/search/dataset/0037918/Sustainable-Development-Goals)
2. Desarrollo Global: [World Development Indicators](https://datacatalog.worldbank.org/search/dataset/0037712/World-Development-Indicators)
3. Gobernanza: [Worldwide Governance Indicators](https://datacatalog.worldbank.org/search/dataset/0038026/Worldwide-Governance-Indicators) *(Selección, monitoreo y reemplazo de gobiernos, capacidad para formular políticas efectivas y respeto por las instituciones)*

## Contenido Destacado
- Importación de librerías.
- Descarga de datos crudos desde el catálogo del Banco Mundial.
- Limpieza e imputación de datos, asegurando conjuntos libres de valores NaN significativos.
- Consolidación y normalización de datos para facilitar el análisis comparativo.
- Análisis estadístico, modelos de regresión y análisis de componentes principales.
- Visualización de resultados y discusión sobre las relaciones entre indicadores de gobernanza y el desarrollo económico.

## Uso
1. Clona este repositorio en tu máquina local.
2. Abre el cuaderno interactivo en tu entorno preferido de Jupyter.
3. Sigue las instrucciones paso a paso para ejecutar el pipeline y analizar los resultados.
